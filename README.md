# 여비 (Yeobi)

출장 교통 영수증(KTX/SRT/고속버스 등) 사진·PDF를 업로드하면 Claude Vision이 판독해서
e-사람에 올릴 **정산신청 리스트(xlsx)**와 **증빙 PDF**를 자동으로 만들어주는 챗봇.

## 실행

```bash
# 1. API 키 설정
cp .env.example .env
# .env 파일을 열어 OPENAI_API_KEY 를 넣는다

# 2. 실행
uv run streamlit run app.py
```

브라우저가 `http://localhost:8501` 로 열린다.

## 사용 흐름

1. 왼쪽 사이드바에서 영수증 파일(여러 장 가능) 업로드.
2. **"N개 영수증 분석"** 버튼 클릭 → Vision 모델이 금액·일자·구간을 추출.
3. 채팅창에 **"정산신청 리스트 만들어줘"** 입력.
4. 오른쪽 패널에서 표 확인 후 **xlsx / PDF** 다운로드.

## 구조

```
app.py                 # Streamlit 진입점 (채팅 UI + 다운로드)
src/yeobi/
  schema.py            # Receipt / ReceiptExtraction Pydantic 모델
  extractor.py         # Claude Vision 호출 → 구조화 추출
  exporter.py          # xlsx 생성 · 영수증 PDF 병합
```

1. 현황 및 문제점

ㅇ 서울 등 근무지외 출장시 기차, 버스 등 영수증 처리가 지나치게 번거롭고 종이와 프린트 잉크 낭비

    * 출장시 KTX, SRT, 고속버스 표를 사비로 결제 후 매달 말 당월 영수증을 각 사이트에 일일이 접속하여 출력 후 과 주무관님께 드리면 주무관님은 e사람 정산신청 리스트에서 영수증에 적힌 금액을 일일이 입력 후 리스트를 출력하고 여기에 종이 영수증을 붙여 과장의 결재 사인을 받아 pdf 파일로 변환 후 각국 총괄과에 제출

2. 개선방향

ㅇ 재경부 AI 여비몬을 개발하여 영수증 처리과정 대폭 간소화

    * 광주광역시는 여비몬 개발하여 운영중 : 출장자가 핸드폰으로 영수증을 촬영해서 올리면 출장내역과 영수증을 자동으로 연결해 전자 증빙이 이뤄져 종이없는 행정 구현. 정산 완료되면 정산 내역과 증빙자료가 통합된 여비정산 결과보고서가 pdf 형태로 즉시 생성되어 별도 문서작업 없이 결재까지 가능

3. 세부 구현방안 (가능하면 광주 여비몬과 최대한 유사하게 개발)

ㅇ ①핸드폰 또는 ②외부망 PC에 여비몬 설치

ㅇ 여비몬은 챗GPT 형태(대화창)로 구성하되 사진 또는 pdf 파일을 첨부할 수 있도록 구성

ㅇ 여비몬에 영수증 파일(사진 또는 pdf)들을 한꺼번에 첨부한 다음 “정산신청 리스트 만들어줘”라고 채팅창에 입력하면 여비몬이 e사람 양식에 호환 가능한 상태의 리스트(영수증도 포함) 생성

ㅇ 이 리스트 파일만 주무관님께 제출하고 주무관님은 과원들의 리스트 파일을 e사람에 올리기만 하면 양식에 자동 반영 (연말정산 파일을 e사람에 올리기만 하면 자동 반영되는 원리)
